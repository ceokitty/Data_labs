# Лабораторна робота №5
## Наука про дані: обмін результатами та початковий аналіз

### Мета роботи
Метою роботи є створення інтерактивного веб-додатку для аналізу даних із використанням бібліотеки Streamlit, що дозволяє здійснювати фільтрацію, візуалізацію та порівняння часових рядів.

---

### Опис роботи
У ході виконання лабораторної роботи було використано дані з лабораторної роботи №2, які представлені у вигляді окремих CSV-файлів для кожної області України.

На першому етапі ці файли були об’єднані в єдиний датасет `final_vhi.csv` за допомогою Python-скрипта `prepare_data.py`. До кожного запису було додано інформацію про область.

На другому етапі було створено веб-додаток із використанням бібліотеки Streamlit для інтерактивного аналізу даних.

---

### Функціонал додатку
Розроблений додаток дозволяє:

- обирати часовий ряд (VCI, TCI або VHI);
- обирати область для аналізу;
- задавати інтервал тижнів;
- задавати інтервал років;
- скидати всі фільтри до початкового стану;
- переглядати відфільтровані дані у вигляді таблиці;
- будувати графік динаміки обраного показника;
- порівнювати значення показника між областями;
- сортувати дані за зростанням або спаданням.


---

### Вимоги до середовища
- Python 3.12
- pip

---

### Встановлення залежностей
```bash
pip install -r requirements.txt
```
### Підготовка даних
```bash
python prepare_data.py
```

### Запуск додатку 
```bash
streamlit run app.py
```

### Результат
Після запуску відкривається веб-додаток у браузері, де користувач може інтерактивно аналізувати дані, змінювати параметри фільтрації та переглядати графіки.

### Висновок
У результаті виконання лабораторної роботи було створено інтерактивний інструмент для аналізу даних. Використання бібліотеки Streamlit дозволило швидко реалізувати веб-інтерфейс та забезпечити зручну взаємодію з даними.
