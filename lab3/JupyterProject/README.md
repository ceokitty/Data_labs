# Лабораторна робота №3  
## Візуалізація даних

### Опис
У межах лабораторної роботи було виконано завантаження, очищення та візуалізацію даних за допомогою Python.  
Для аналізу використано датасет **Adult Income** з репозиторію **UCI Machine Learning Repository**.  
У роботі проведено очищення даних, обробку пропущених значень та побудовано набір графіків для аналізу розподілу даних і залежностей між змінними.

---

## Вимоги до системи
- **Операційна система:** Windows / macOS / Linux
- **Python:** 3.12 або новіше
- **pip:** актуальна версія менеджера пакетів Python
- **Jupyter Notebook** або **PyCharm Professional** з підтримкою Jupyter

## Встановлення залежностей

1. Клонувати репозиторій:
```bash
git clone <посилання_на_репозиторій>
cd <назва_папки_проєкту>
```

2. Створити та активувати віртуальне середовище(macOS / Linux):
```bash
python3 -m venv .venv
source .venv/bin/activate
```
3. Встановити залежності:
```bash
pip install -r requirements.txt
```
