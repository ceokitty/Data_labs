# Лабораторна робота №6

## Обробка відео з камери за допомогою OpenCV та C++

---

## Опис проєкту

Даний проєкт реалізує програму для роботи з вебкамерою за допомогою бібліотеки OpenCV.  
Програма зчитує відео з камери в реальному часі, виконує обробку кадрів та відображає результат у вікні.

Проєкт побудований за модульною архітектурою з використанням окремих класів для:

- отримання кадрів з камери;
- обробки клавіш;
- обробки зображень;
- відображення результату.

---

## Використані технології

- C++
- OpenCV

- GCC / Clang
- CLion

---

## Функціональні можливості

Програма підтримує декілька режимів обробки зображення:

| Клавіша | Режим             |
| ------- | ----------------- |
| 0       | Звичайний режим   |
| 1       | Інверсія кольорів |
| 2       | Відтінки сірого   |
| 3       | Gaussian Blur     |
| 4       | Canny Filter      |
| 5       | Sobel Filter      |
| 6       | Бінаризація       |
| q / ESC | Вихід з програми  |

## Принцип роботи програми

1. Камера ініціалізується через клас `CameraProvider`.
2. Отриманий кадр передається до `FrameProcessor`.
3. Залежно від натиснутої клавіші застосовується певний фільтр.
4. Клас `Display` відображає результат у вікні.
5. Клас `KeyProcessor` обробляє натискання клавіш та змінює режим роботи програми.
