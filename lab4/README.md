# Лабораторна робота №4  
## Візуалізація даних 2  

### Мета роботи  
Розробити програму для побудови гармонічного сигналу з накладеним шумом, а також реалізувати інтерактивний інтерфейс для зміни параметрів гармоніки, шуму та параметрів фільтрації.  

### Використані бібліотеки  
- Python 3.x  
- numpy  
- matplotlib  
- scipy  

### Запуск програми  
Для встановлення залежностей виконайте:  

```bash
pip install -r requirements.txt
```

### Опис програми
Програма будує гармонічний сигнал виду:
y(t) = A * sin(ωt + φ)

До сигналу додається випадковий шум із заданими параметрами.
Користувач може змінювати параметри сигналу та шуму в реальному часі за допомогою інтерактивного інтерфейсу.

Програма відображає два графіки:

* верхній — чиста та зашумлена гармоніка
* нижній — чиста та відфільтрована гармоніка

## Функціонал
Програма містить:

* слайдери для зміни амплітуди, частоти, фази
* слайдери для параметрів шуму (середнє значення та дисперсія)
* слайдер для керування фільтрацією (Cutoff Frequency)
* checkbox для увімкнення/вимкнення шуму
* кнопку Reset для відновлення початкових параметрів

## Інструкція для користувача
1. Запустіть програму.
2. Змінюйте параметри сигналу за допомогою слайдерів.
3. Використовуйте Show Noise для увімкнення або вимкнення шуму.
4. Змінюйте Cutoff Frequency для керування фільтрацією.
5. Спостерігайте зміни на графіках у реальному часі.
6. Натисніть Reset, щоб повернути початкові значення.

## Особливості реалізації
* шум генерується лише при зміні його параметрів
* при зміні гармоніки шум не перегенеровується
* використано Butterworth-фільтр для очищення сигналу
* відфільтрований сигнал наближений до чистої гармоніки

## Висновок
У ході виконання лабораторної роботи було створено інтерактивну програму для візуалізації гармонічного сигналу з шумом.
Реалізовано зміну параметрів сигналу та шуму, а також фільтрацію, що дозволяє порівняти чисту, зашумлену та відфільтровану гармоніки.
