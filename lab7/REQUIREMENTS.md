# Лабораторна робота №7  
## Computer Vision та багатопотоковість у C++

### Мета роботи

Метою лабораторної роботи було ознайомлення з можливостями бібліотеки OpenCV для задач Computer Vision та використання багатопотоковості в C++.  
У ході роботи було реалізовано систему детекції обличчя за допомогою нейронної мережі ResNet-10 SSD та винесено процес обробки кадрів у окремий потік для підвищення плавності роботи відео.

---

# Опис роботи програми

Програма отримує відео з вебкамери через `cv::VideoCapture`.  
Кожен кадр передається у клас `FaceDetector`, який працює у фоновому потоці.

Для пошуку облич використовується модуль `cv::dnn` та попередньо навчена нейронна мережа ResNet-10 SSD.

Після виконання детекції координати знайдених облич передаються назад у головний потік, де навколо облич малюється прямокутник.

---

# Архітектура програми

Проєкт складається з декількох основних файлів:

# Робота багатопотоковості

## Main Thread (основний потік)

Основний потік:
1. Отримує кадр з камери.
2. Передає копію кадру у `FaceDetector`.
3. Отримує останні координати знайдених облич.
4. Малює рамки на поточному кадрі.
5. Відображає відео.

---

## Worker Thread (фоновий потік)

Фоновий потік:
1. Очікує новий кадр.
2. Запускає нейронну мережу.
3. Виконує детекцію облич.
4. Оновлює координати знайдених облич.
5. Повторює цикл.

---

# Чому використовується багатопотоковість

Інференс нейронної мережі є досить важкою операцією та може займати десятки або сотні мілісекунд.

Якщо виконувати детекцію у головному циклі програми, відео почне:
- зависати;
- втрачати FPS;
- працювати ривками.

Тому детекція була винесена у окремий потік.

Завдяки цьому:
- інтерфейс залишається плавним;
- відео не фрізить;
- FPS залишається стабільним.

---

# Захист спільних даних

Для безпечної роботи потоків використовується:

## std::mutex

Використовується для захисту:
- кадру камери;
- координат знайдених облич.

М’ютекс не дозволяє двом потокам одночасно змінювати одні й ті самі дані.

---

## std::atomic<bool>

Використовується як прапорець для безпечної зупинки фонового потоку.
