# Лабораторна робота №2. Наука про дані: підготовчий етап

## Опис завдання
1. Завантаження даних (VHI-індекс) за допомогою `urllib`, очистка даних у `pandas dataframe` та створення функцій для вибірок.
2. Аналіз датасету *Individual Household Electric Power Consumption*, очистка даних, формування вибірок, підрахунок коефіцієнтів Пірсона та Спірмена, One Hot Encoding.

## Вимоги до системи
- **ОС:** macOS (Apple Silicon M1)
- **Інтерпретатор:** Python 3.x

## Інструкція із запуску
1. Активувати віртуальне середовище: `source venv/bin/activate`
2. Встановити залежності: `pip install -r requirements.txt`
3. Запустити середовище: `jupyter notebook`
